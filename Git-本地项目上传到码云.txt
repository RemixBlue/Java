上传：
1，首先在码云上新建一个项目
2，进入本地项目的文件夹，点击鼠标右键，选择git bash here
然后在窗口输入 git init  这时候文件夹会多出一个.git文件夹，看不到文件夹的，点击鼠标右键选择“显示不显示隐藏的文件”就可以看到这个文件夹了
3、输入git remote add origin + 仓库路径
4、输入 git pull origin master 命令，将码云上的仓库pull到本地文件夹
5、输入 git add .  
6、输入 git commit -m '新添加的文件内容描述'
7、输入 git push origin master ，将本地仓库推送到远程仓库
8、上传成功！
